Overview

MindMatch is a fun, easy-to-use app with multiple puzzle games in one place. Inspired by classic puzzle games, it packs three mini-games (Pattern Memory, Mastermind, and Jigsaw) with different difficulty levels. It’s designed for short, skill-based sessions you can play anywhere, with logic-based challenges that test memory, focus, and reasoning. Daily challenges keep the loop fresh, and sound/animation touches make gameplay more exciting. Firebase-backed profiles, leaderboards, and creation tools tie it all together.

Key Features

Profiles
  • Creation of profiles with editable display name and bio so players feel personal ownership.
  • Saved progress tied to each profile across all puzzles for seamless pick-up-and-play.
Social
  • Leaderboard per puzzle type to track high scores and completion times.
  • AI-generated daily challenges that keep the experience fresh every day.
Content
  • Puzzle creation tools (Jigsaw uploads, Mastermind configuration) for user-generated content.
  • Puzzle library with multiple puzzle types for variety and replayability.

Challenges

  • Firebase technical limitation: planned to store Jigsaw images in Firebase for dynamic downloads, but billing limits blocked it; storing images locally instead.
  • Creating daily challenges with Gemini: validating Gemini responses to keep parameters within Mastermind constraints, then saving the generated puzzle to Firestore for all users.
  • Firebase data conversion: puzzle data needed conversion to Firebase-friendly formats (Duration to seconds, Instant to Timestamp, enums to strings) via custom converters.

Tools

  • Kotlin: primary language for app logic and UI (Jetpack Compose) in Android Studio; all screens and data models are Kotlin-first.
  • Firebase: authentication and data storage for puzzles, progress, leaderboards, and daily challenges; Firestore/Storage back the live data.
  • Android Studio: main IDE for development, debugging, and device/emulator testing.
  • Gemini: generates unique daily Mastermind puzzles saved to Firestore and surfaced to all users.

Build & Run

  • Requirements: Android Studio Ladybug+, JDK 17, Android SDK 26+.
  • Sync and build: ./gradlew assembleDebug
  • Run on device/emulator via Android Studio or ./gradlew installDebug.
  • Generate API docs: ./gradlew :app:dokkaHtml → open app/build/dokka/html/index.html.
